It's not rare that you find tools that apply for other countries but none exist for Ethiopian contexts.
When dealing with financial transactions, manually calculating VAT and withholding tax can be tedious and error-prone.
This program takes a post-VAT or pre-VAT price, calculates the pre-VAT/post-VAT amount, and applies withholding tax if 
thresholds are met (10,000 for products, 3,000 for services), displaying then both the net payable amount and the withheld amount
